# Custom Fig Plugins

**Prerequisite: You must have fig-autocomplete clone locally**

https://fig.io/docs/getting-started

## Plugins Offered

1. alias-finder
2. gi
3. node-docs

## Use These Plugins

1.  Add the PATH_TO_FIG_REPO to the .env file
2.  Run copy.sh
3.  From your fig-autocomplete repo, run "npm run dev"
